# dynamic-image-grid

A masonry grid filled with images and using beautiful-dnd react library to reorder images. The items in the grid should be easy to drag and drop and reorder automatically and dynamically with beautiful animations for various states

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dynamic-image-grid.git
cd dynamic-image-grid
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
